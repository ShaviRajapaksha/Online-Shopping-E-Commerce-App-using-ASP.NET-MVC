# Online Shopping E-Commerce App (ASP.NET MVC)

The **Online Shopping E-Commerce Application** is a full-featured web platform built using the **ASP.NET MVC framework**, designed to provide customers with a smooth online shopping experience while offering administrators powerful tools for managing products, orders, and users. The system follows a modular, scalable architecture and applies modern web development practices to ensure high performance, maintainability, and security.

---

## üöÄ Key Features

### **1. User Module**
- User registration & authentication (ASP.NET Identity)
- User profile management
- Secure login & password recovery
- Role-based access control (Admin / Customer)

### **2. Product Management**
- Product catalog with categories & subcategories
- Detailed product pages (images, price, description, stock)
- Product search & filtering (category, price, keyword)
- Admin panel CRUD operations:
  - Products
  - Categories
  - Brands

### **3. Shopping Cart**
- Add, update, and remove products
- Persistent cart tied to user accounts
- Display total amount, tax, and quantity
- Supports guest carts (optional)

### **4. Checkout & Orders**
- Address & payment information form
- Order confirmation page
- Order tracking status (Pending, Shipped, Delivered)
- Invoice generation & email notification (optional)

### **5. Admin Dashboard**
- Manage customers, products, orders
- View sales statistics & reports
- Inventory management with stock alerts

### **6. Additional Optional Features**
- Payment gateway integration (Stripe, PayPal)
- Rating & reviewing system
- Wishlist / Favorites
- Coupon codes / Discounts
- Product recommendations
- Responsive UI (Bootstrap)
